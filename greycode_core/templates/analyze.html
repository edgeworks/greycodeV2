Analyze endpoint export

Upload a Splunk JSON export for a single endpoint and compare Sysmon 1, 3 and 22 entries against the existing Greycode dataset. Imported data is not stored in Greycode.

Splunk query hint
Use a normalized export with the fields below and then export as JSON. Greycode also accepts the Splunk wrapper format with {"result": {...}}. 
index=wef Computer="HOSTNAME" (EventCode=1 OR EventCode=3 OR EventCode=22)
| eval sha256=coalesce(
    sha256,
    mvindex(split(mvindex(split(Hashes,"SHA256="),1),","),0)
  )
| eval CommandLine=coalesce(CommandLine, process_command_line, "")
| eval Image=coalesce(Image, image, process_path, "")
| eval DestinationIp=coalesce(DestinationIp, dest_ip, "")
| eval DestinationPort=coalesce(DestinationPort, dest_port, "")
| eval QueryName=coalesce(QueryName, query_name, "")
| eval Computer=coalesce(Computer, host)
| table EventCode _time Computer Image CommandLine sha256 DestinationIp DestinationPort QueryName
| sort 0 _time
Supported JSON
Accepted formats: plain JSON objects, JSON arrays, NDJSON, and Splunk export lines with {"preview":false,"result":{...}}.
